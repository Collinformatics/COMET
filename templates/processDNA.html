COmprehensive Motif Evaluation Toolkit

Process DNA

This program extracts DNA sequences found between two flanking sequences. If the extracted DNA sequence matches the specified sequence length then the sequence will be translated. Sequences not containing a stop codon or unknown AA will be saved.

Uploaded files must be formatted as FASTQ or FASTA.
Output files can be used for filtering.

? Experimental substrates.
Acceptable file extensions: .fasta, .fasta.gz, .fa, .fa.gz, .fastq, .fastq.gz, .fq, .fq.gz
? Background substrates.
Acceptable file extensions: .fasta, .fasta.gz, .fa, .fa.gz, .fastq, .fastq.gz, .fq, .fq.gz
? Extract DNA sequence between these sequences.
Ex: 5'seq-substrate-3'seq
? Number of AAs in the substrate.
? Drop substrates with fewer counts than this value.
? Minimum quality score of acceptable DNA reads.
? Select substrates with a given AA at a defined residue in the sequence.
? Exclude substrates with a given AA at a defined residue in the sequence.
