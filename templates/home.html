COmprehensive Motif Evaluation Toolkit

Read Me

COMET is a mulituse toolkit that can be used for processing enzyme profiling data.

The available tools can:
- Evaluate and extract protein sequences from FASTQ or FASTA files generated from sequencing methods such as NGS.
- Evaluate amino acid (AA) enrichment in an experimental (Exp) dataset relative to a background (Bg) set.
- Filter randomized sequences to find Recognition Sites and evaluate the Substrate Profile.
- Combine multiple Substrate Profiles, and evaluate convergence.



Process DNA:

Upload a FASTQ or FASTA file to translate the DNA sequences. The prevalence of amino acids in your datasets will be evaluated. Both the counts matrix, and protein sequences can then be downloaded and used for subsequent analyses.



Filter AA:

To apply a filter to the data, select which residues will be fixed, or excluded from a given position in the sequence.

If you don't know which AAs to fix, then don't add a filter to plot the raw data. This will indicate favorable residues.



Filter Motif:

By applying and filter to a randomized set of substrates, and then evaluating the frequencies of each amino acid in that set, we can determine entropy scores (∆S) at each position in the sequence. This reveals the Recognition Site, or the sites that are important for substrate recognition.



Combine Profiles:

Combine the Recognition Sites from multiple Substrate Profiles that were obtained from the Filter Motif analysis.



Equations:

By comparing the relative frequency (RF) of finding a given residue in the Experimental and Background datasets, the Enrichment Score (ES) of each residue can be evaluated.

ES = log2(RFAA Exp / RFAA Bg)

The RF of each AA in the Experimental set are used to determine the Positional Entropy (∆S) at each position in the sequence.

∆S = SMax - SShannon = log2(20) - ∑(-RFAA * log2(RFAA))
