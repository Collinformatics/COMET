COmprehensive Motif Evaluation Toolkit

Read Me

COMET is a mulituse toolkit that can be used for processing enzyme profiling data.

The available tools can:
- Evaluate and extract protein sequences from FASTQ or FASTA files generated from sequencing methods such as NGS.
- Evaluate amino acid (AA) enrichment in an experimental (Exp) dataset relative to a background (Bg) set.
- Filter randomized sequences to find Recognition Sites and evaluate the Substrate Profile.
- Combine multiple Substrate Profiles, and evaluate convergence.



Process DNA:

Upload a FASTQ or FASTA file to translate the DNA sequences. The prevalence of amino acids in your datasets will be evaluated. Both the counts matrix, and protein sequences can then be downloaded and used for subsequent analyses.



Filter AA:

To apply a filter to the data, select which residues will be fixed, or excluded from a given position in the sequence. If you don't know which AAs to fix, then don't add a filter and plot the raw data. This will indicate favorable residues.



Filter Motif:

By evaluating the amino acid probabilities in the Exp set, we can determine entropy scores (∆S) at each position in the sequence.



Equations:

By comparing the probability of finding a given residue in the Exp and Bg datasets, the Enrichment Score (ES) of each residue can be evaluated.

ES = log2(probAA Exp / probAA Bg)

The AA probabilities in the Exp set are used to determine the change in entropy (∆S) at a given position in the sequence.

∆S = SMax - SShannon = log2(20) - ∑(-probAA * log2(probAA))
